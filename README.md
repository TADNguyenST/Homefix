# HomeFix

HomeFix is a home appliance repair booking system.

## Modules planned

- Authentication and profile management
- Customer address management
- Service category management
- Service and device type management
- Voucher management
- Booking, technician assignment, quotation, payment, notification and AI support in later iterations

## Project structure

```text
client/   React frontend
server/   Express backend
docs/     Backlog and git plan
```

## Run locally

```bash
cd server
npm install
npm run dev
```

```bash
cd client
npm install
npm run dev
```
